# 🏥 Medical Cost Prediction using Linear Regression (From Scratch)

An educational Machine Learning project that implements **Linear Regression from scratch using only NumPy**, without relying on machine learning libraries such as scikit-learn.

The goal of this project is to understand the mathematics behind Linear Regression, Data Cleaning, Feature Engineering, and Gradient Descent by building everything manually.

---

## 🚀 Features

- Data Cleaning
- Missing Values Handling
- Outlier Detection & Repair
- Duplicate Removal
- Inconsistent Data Correction
- Feature Encoding
- Feature Normalization (Z-score)
- Linear Regression implemented from scratch
- Gradient Descent Optimization
- Cost Function Monitoring

---

## 📂 Dataset

Medical Cost Personal Dataset

Target Variable:

- `charges`

Features:

- age
- sex
- bmi
- children
- smoker
- region

---

## 🛠 Technologies

- Python
- NumPy
- Pandas

---

## 📈 Workflow

1. Load Dataset
2. Simulate Data Problems
   - Missing Values
   - Outliers
   - Duplicate Rows
   - Inconsistent Values
3. Clean the Dataset
4. Encode Categorical Features
5. Normalize Features
6. Train Linear Regression using Gradient Descent
7. Predict Medical Insurance Charges

---

## 📚 Learning Objectives

This project focuses on understanding:

- Data preprocessing
- Gradient Descent
- Linear Regression Mathematics
- Matrix Operations using NumPy

instead of using ready-made machine learning models.

---

## ⚠️ Note

This project was built for educational purposes.

The primary objective is learning and implementing Linear Regression manually rather than achieving production-level prediction accuracy.
